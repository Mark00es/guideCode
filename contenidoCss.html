Hojas de Estilo en Cascada es el lenguaje de estilos utilizados para describir la presentación de documentos HTML.
CSS es utilizado para diseñar y dar estilo a las páginas web, por ejemplo, alterando la fuente, color, tamaño y espaciado del contenido, dividirlo en múltiples columnas o agregar animaciones y otras características decorativas.

ELEMENTOS CSS

Nombre Descripción Formato Ejemplo
Color Se inserta el color en ingles color: red;
 
rgb Mezcla intensidades de los colores primarios rgb(red, green, blue)
 
rgba El valor a especifica la transparencia rgba(red, green, blue, Opacity)
 
hsl Define el color saturacion y la iluminacion hsl(hue, saturation, lightness)
 
hsla Es igual que la funcion hsl pero con la opacidad hsla(hue, saturation, lightness, Opacity)
 

Selectores Css

Nombre Descripción Formato
Selector Universal Selecciona todos los elementos. *
Selector de Tipo Permite seleccionar elementos usando el nombre del elemento. div
Clase Permite seleccionar elementos en función de los valores del atributo class .NombreClase
Id Permite seleccionar elementos en función de los valores del atributo id unicos #NombreId
tipo atributos Un tipo con un atributo "attr". Tipo[attr]
tipo atributos Un elemento selector cuyo valor de atributo "attr" es exactamente igual a value. selector[attr=value]
tipo atributos Un elemento selector cuyo valor de atributo "attr" es una lista de valores separados por espacios en blanco, uno de los cuales es exactamente igual a value. selector[attr~=value]
tipo atributos Un elemento selector cuyo valor de atributo "attr" comienza exactamente con la cadena value. selector[attr^=value]
tipo atributos Un elemento selector cuyo valor de atributo "attr" termina exactamente con la cadena value. selector[attr$=value]
tipo atributos Un elemento selector cuyo valor de atributo "attr" contiene la subcadena value. selector[attr*=value]
tipo atributos Un elemento selector cuyo atributo "attr" tiene una lista de valores separados por guiones que comienzan (desde la izquierda) con "en". selector[attr|="en"]
Pseudoclases Un elemento selector, raíz del documento. selector:root
Pseudoclases Un elemento selector, el n-ésimo hijo de su padre. selector:nth-child(n)
Pseudoclases Un elemento selector, el n-ésimo hijo de su padre, contando desde el último. selector:nth-last-child(n)
Pseudoclases Un elemento selector, el n-ésimo hermano de su tipo. selector:nth-of-type(n)
Pseudoclases Un elemento selector, el n-ésimo hermano de su tipo, contando desde el último. selector:nth-last-of-type(n)
Pseudoclases Un elemento selector, primer hijo de su padre. selector:first-child
Pseudoclases Un elemento selector, último hijo de su padre. selector:last-child
Pseudoclases Un elemento selector, primer hermano de su tipo. selector:first-of-type
Pseudoclases Un elemento selector, último hermano de su tipo. selector:last-of-type
Pseudoclases Un elemento selector, único hijo de su padre. selector:only-child
Pseudoclases Un elemento selector, único hermano de su tipo. selector:only-of-type
Pseudoclases Un elemento selector que no tiene hijos (incluidos los nodos de texto). selector:empty
Pseudoclases de enlace Siendo un elemento selector el ancla fuente de un hipervínculo cuyo destino aún no ha sido visitado. selector:link
Pseudoclases de enlace Siendo un elemento selector el ancla fuente de un hipervínculo cuyo destino aún no ha sido visitado. selector:visited
Pseudoclases Accion Un elemento selector durante ciertas acciones del usuario. selector:active
Pseudoclases Accion Un elemento selector durante ciertas acciones del usuario. selector:hover
Pseudoclases Accion Un elemento selector durante ciertas acciones del usuario. selector:focus
Pseudoclase objetivo Un elemento selector es el destino del URI de referencia. selector:target
Lang() Un elemento de tipo selector en el idioma "fr". selector:lang(fr)
Enabled Un elemento de interfaz de usuario selector esta habilitado. selector:enabled
Disabled Un elemento de interfaz de usuario selector esta deshabilitado. selector:disabled
Checked Un elemento tipo la interfaz de usuario que está marcado. selector:checked
Primera Linea La primera línea formateada de un elemento selector. selector::first-line
Primera letra La primera letra formateada de un elemento selector. selector::first-letter
Antes Contenido generado antes de un elemento selector. selector::before
Despues Contenido generado después de un elemento selector. selector::after
