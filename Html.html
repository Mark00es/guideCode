HTML

Es un archivo de texto plano escrito usando HTML
HTML nos permite crear documentos interconectados. Significa que podemos saltar de un documento a otro por medio de enlaces que el usuario activa.
HTML es un lenguaje de marcado porque permite definir la estructura y contenido del documento usando marcas, es decir etiquetas.
