ELEMENTOS HTML

Elemento Descripción
<head> Información general (metadatos) acerca del documento
<body> Es el cuerpo del documento HTML
<html> Representa la raiz del documento
<div> Elemento generico
<p> Distribuye el texto en párrafos
<input> Crea controles interactivos para formularios basados en la web
<span> Contenedor genérico en línea para redactar contenido, que no representa inherentemente nada
<nav> Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación
<aside> Representa una parte de un documento cuyo contenido solo está indirectamente relacionado con el contenido principal del documento
<button> Elemento interactivo activado por un usuario con un mouse, teclado, dedo, comando de voz u otra tecnología de asistencia
<footer> representa un pie de página
<main> Representa el contenido dominante de un documento.
<a> crea un hipervínculo a páginas web
<article> composición independiente en un documento
<title> define el título del documento
<wbr> Representa una oportunidad de salto de palabra
<sup> Especifica el texto en línea que se mostrará como superíndice únicamente por motivos tipográficos
<sub> Especifica texto en línea que debe mostrarse como subíndice únicamente por motivos tipográficos
<strong> Indica que sus contenidos tienen una gran importancia, seriedad o urgencia
<em> Marca el texto que tiene énfasis
<mark> Representa texto que está marcado o resaltado para fines de referencia o notación
<q> Indica que el texto adjunto es una breve cita en línea
<blockquote> Indica que el texto adjunto es una cita extendida
<cite> Se usa para describir una referencia a un trabajo creativo citado y debe incluir el título de ese trabajo
<dfn> Se utiliza para indicar el término que se define dentro del contexto de una frase u oración de definición
<abbr> Representa una abreviatura o un acrónimo
<small> Representa comentarios secundarios y letra pequeña
<ins> representa un rango de texto que se ha agregado a un documento
<del> Representa un rango de texto que se ha eliminado de un documento
<s> tachar texto
<code> muestra su contenido con estilo para indicar que el texto es un fragmento corto de código de computadora
<samp> Se utiliza para representar la salida de muestra
<kbd> Representa un tramo de texto en línea que indica la entrada de texto del usuario desde un teclado
<pre> Representa texto preformateado que debe presentarse exactamente como está escrito en el archivo HTML
<address> Indica que el HTML adjunto proporciona información de contacto para una persona o personas
<hr> Representa una ruptura temática entre elementos a nivel de párrafo
<h1>-<h6> Representan seis niveles de encabezados de sección
<table> Representa una tabla
