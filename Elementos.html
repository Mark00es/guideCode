ELEMENTOS HTML

Elemento Descripción Documentación
html Representa la raiz del documento Documentacion
div Elemento generico Documentacion
table Representa una tabla Documentacion
p Distribuye el texto en párrafos Documentacion
input Crea controles interactivos para formularios basados en la web Documentacion
head Información general (metadatos) acerca del documento Documentacion
span Contenedor genérico en línea para redactar contenido, que no representa inherentemente nada Documentacion
button Elemento interactivo activado por un usuario con un mouse, teclado, dedo, comando de voz u otra tecnología de asistencia Documentacion
body Es el cuerpo del documento Documentacion
footer representa un pie de página Documentacion
a crea un hipervínculo a páginas web Documentacion
article composición independiente en un documento Documentacion
title define el título del documento Documentacion
